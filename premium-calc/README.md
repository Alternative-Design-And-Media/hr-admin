# Prémium Kalkulátor – Outline Central Europe Kft.

Bérszámfejtői eszköz a havi prémium kiszámításához a Prémium Szabályzat (2026.03.01.) alapján.

## Lokális fejlesztés

```bash
npm install
npm run dev
```

## Deploy – Cloudflare Pages

**Build beállítások:**
- Root directory: `premium-calc`
- Framework preset: `Vite`
- Build command: `npm run build`
- Build output directory: `dist`
- Node version: `18`
